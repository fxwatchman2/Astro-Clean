SVG Pattern Visual Test

Dots Pattern (id="dots-pattern")

Small dots. Modify circle attributes (cx, cy, r, fill) in the pattern definition to change appearance.

Diagonal Hatch Pattern (id="diagonal-hatch-pattern")

Diagonal lines. Modify line attributes (stroke, stroke-width) and patternTransform (rotate angle) to change appearance.

Cross Hatch Pattern (id="cross-hatch-pattern")

Crossed diagonal lines. Modify line attributes (stroke, stroke-width) to change appearance.

Note: The fill and stroke colors within the <pattern> definitions control the pattern's appearance. For example, fill="rgba(0,0,0,0.5)" is semi-transparent black. You can change these to any valid SVG color (e.g., "blue", "#FF0000", "rgba(255,0,0,0.7)").
